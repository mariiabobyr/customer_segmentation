### customer_segmentation

## Project Overview
This project analyzes an e-commerce dataset to divide customers into clusters

## Objectives
- Identify different customer groups

## Tools
- Python
- Pandas
- Scikit_learn
- Matplotlib
- Jupyter Notebook

## Conclusions
Three customer segments were identified:
- Frequent Buyers
- Low-value customers
- VIP customers

## Recommendations
- Focus on retaining VIP customers through loyalty programs
- Increase average order value for frequent buyers via targeted promotions
- Use cost-efficient marketing strategies for low-value customers

## Project Structure
- customer_segmentation.ipynb – main notebook
- dataset *online_retail.csv* – source data

## Dataset
The dataset is available on Kaggle: https://www.kaggle.com/datasets/ulrikthygepedersen/online-retail-dataset
